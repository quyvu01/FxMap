# Changelog v9.0.0

All notable changes in this release.

## [9.0.0] - 2026-02-11

### Breaking Changes

This is a **major release** with comprehensive naming improvements across the entire FxMap ecosystem. All changes are rename-only — no behavioral or functional changes.

### Migration Notes

**From v8.3.0 to v9.0.0:**

All changes are renames. No behavioral changes, no new features, no removed features. Use find-and-replace to update your codebase:

**Step 1: Update `using` statements**

```bash

find . -name "*.cs" -exec sed -i '' 's/using FxMap\.ApplicationModels/using FxMap.Models/g' {} +
find . -name "*.cs" -exec sed -i '' 's/using FxMap\.ObjectContexts/using FxMap.PropertyMappingContexts/g' {} +
find . -name "*.cs" -exec sed -i '' 's/using FxMap\.Externals/using FxMap.PublicContracts/g' {} +
find . -name "*.cs" -exec sed -i '' 's/using FxMap\.Cached/using FxMap.MetadataCache/g' {} +
find . -name "*.cs" -exec sed -i '' 's/using FxMap\.InternalPipelines/using FxMap.BuiltInPipelines/g' {} +
find . -name "*.cs" -exec sed -i '' 's/using FxMap\.Statics/using FxMap.Configuration/g' {} +

# Transport namespace changes
find . -name "*.cs" -exec sed -i '' 's/using FxMap\.RabbitMq\.ApplicationModels/using FxMap.RabbitMq.Configuration/g' {} +
```
